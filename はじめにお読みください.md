# Column Select — Installation Guide

A Chrome extension that brings **rectangular (column) selection** to any web page — just like Sakura Editor.  
Works in `textarea` and `contenteditable` fields on Gmail, Yahoo Mail, and virtually any website.

---

## How to install (about 5 minutes)

### 1. Prepare the folder

Unzip this archive somewhere permanent (e.g. inside your `Documents` folder).

> ⚠️ **Do not move or delete the unzipped folder.**  
> The extension loads directly from it — removing the folder breaks the extension.

### 2. Open Chrome Extensions

Type the following into Chrome's address bar and press Enter:

```
chrome://extensions
```

### 3. Enable Developer Mode

Toggle **Developer mode** on (top-right corner of the page).

### 4. Load the extension

Click **"Load unpacked"** and select the **`browser_column_select`** folder.

### 5. Done

If **"矩形選択拡張 (Column Select)"** appears in the list, you're all set.

---

## How to use

| Shortcut | Action |
|---|---|
| **Alt + Drag** | Select a rectangle with the mouse |
| **Alt+Shift+Arrow** / **Ctrl+Alt+Arrow** | Extend rectangle selection with keyboard |
| **Ctrl + C** | Copy selected rectangle |
| **Ctrl + X** | Cut selected rectangle |
| **Delete / Backspace** | Delete selected rectangle |
| **Type a character** | Insert into each selected row (column insert) |
| **Ctrl + H** | Open find & replace dialog |
| **Ctrl + Z** | Undo |
| **Esc** | Cancel selection / close dialog |

> While holding Alt, a **"Col Select"** badge appears at the bottom-right of the screen.

### Ctrl+H replace scope

| State when opening | Scope |
|---|---|
| Rectangle selected | Only the selected rows |
| Text selected | Only the selected text |
| Nothing selected | Entire document body |

---

## Troubleshooting

- **Alt+Shift+Arrow doesn't work**  
  Windows may be using Alt+Shift for language switching. Use **Ctrl+Alt+Arrow** instead.
- **Extension stopped working**  
  Check that the folder has not been moved or deleted.

# 矩形選択拡張 (Column Select) — インストール手順

任意のWebページで、サクラエディタのような **矩形(カラム)選択** ができるようになるChrome拡張機能です。  
Gmail・Yahoo Mail・その他あらゆるサイトの `textarea` や `contenteditable` 編集エリアで動作します。

---

## インストール方法（5分程度）

### 1. フォルダを準備する

このZIPを、消さない場所に解凍してください（例：`ドキュメント` フォルダの中など）。

> ⚠️ **注意:** 解凍したフォルダは消したり移動したりしないでください。  
> 拡張機能はこのフォルダを直接読み込んで動くため、フォルダを消すと動かなくなります。

### 2. Chromeの拡張機能ページを開く

Chromeのアドレスバーに入力してEnter:

```
chrome://extensions
```

### 3. 「デベロッパーモード」をオンにする

画面の**右上**にある「デベロッパー モード」のスイッチをオンにします。

### 4. 拡張機能を読み込む

「**パッケージ化されていない拡張機能を読み込む**」をクリックし、**`browser_column_select` フォルダ** を選びます。

### 5. 完了

一覧に「矩形選択拡張 (Column Select)」が表示されれば成功です。

---

## 使い方

| 操作 | 内容 |
|---|---|
| **Alt + ドラッグ** | マウスで矩形を選択 |
| **Alt+Shift+矢印** / **Ctrl+Alt+矢印** | キーボードで矩形選択を広げる |
| **Ctrl + C** | コピー |
| **Ctrl + X** | カット |
| **Delete / Backspace** | 削除 |
| **文字キー** | 各行の選択左端にカラム挿入 |
| **Ctrl + H** | 置換ダイアログを開く |
| **Ctrl + Z** | 元に戻す |
| **Esc** | 選択解除 / ダイアログを閉じる |

※ Altキーを押している間、画面右下に「Col Select」バッジが表示されます。

### Ctrl+H の置換スコープ

| 状態 | 置換される範囲 |
|---|---|
| 矩形選択中 | 選択した各行の範囲内だけ |
| テキスト選択中 | 選択した部分だけ |
| 何も選択していない | 本文全体 |

---

## うまく動かないとき

- **Alt+Shift+矢印が効かない** → Windowsの言語切り替えと競合している場合があります。**Ctrl+Alt+矢印** を使ってください。
- **拡張機能が消えた/動かない** → 解凍したフォルダを移動・削除していないか確認してください。
