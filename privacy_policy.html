Privacy Policy

Column Select — Chrome Extension

Summary

This extension does not collect, store, transmit, or share any personal data. All processing happens entirely within your browser.

Data collection

Column Select collects no data whatsoever. Specifically:

  • No browsing history is recorded.
  • No text you select, copy, or edit is sent anywhere.
  • No analytics, tracking pixels, or third-party scripts are used.
  • No account or sign-in is required.

Permissions

The extension uses content_scripts to inject a small JavaScript file into web pages you visit. This script only listens for mouse and keyboard events to enable rectangular selection. It does not read page content for any other purpose.

Clipboard

When you use Ctrl+C or Ctrl+X, the selected text is written to your local clipboard using the standard browser Clipboard API. This data never leaves your device.

Third parties

No data is shared with any third party. The extension makes no network requests.

Changes to this policy

If this policy changes, an updated version will be published at the same URL.

Contact

Questions? Open an issue on the GitHub repository or contact via the Chrome Web Store listing.
