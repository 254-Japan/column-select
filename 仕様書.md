# Column Select — Technical Specification

A Chrome extension that enables rectangular (column) selection in `<textarea>` and `contenteditable` editing areas on any web page, inspired by Sakura Editor's block selection feature.

## How it works

1. Hold **Alt** and drag the mouse inside an editable area — a rectangle is highlighted with a blue border and yellow fill.
2. **Ctrl+C** — copies the selected text to the clipboard (rows joined with `\n`).
3. **Ctrl+X** — copies and deletes the selected text (cut).
4. **Delete / Backspace** — deletes the selected text without copying.
5. **Esc** or click elsewhere — clears the selection.

Cut/delete operations use `document.execCommand` so they are undoable with **Ctrl+Z**.

### Keyboard rectangle selection

- Start from the cursor position and extend with **Alt+Shift+Arrow** or **Ctrl+Alt+Arrow**.
- On Windows, Alt+Shift may be captured by the IME language switcher — use Ctrl+Alt+Arrow in that case.
- After extending, the same Ctrl+C / Ctrl+X / Delete / Backspace shortcuts apply.

### Column insert

While a rectangle is selected, typing any character inserts it at the left edge of each selected row.  
IME (CJK) input is also supported — the composed text is inserted upon confirmation.

### Find & Replace (Ctrl+H)

Opens a dialog with scope awareness:
- Rectangle selected → replaces within each selected row only
- Text selected → replaces within the selection only
- Nothing selected → replaces across the entire editable area

## Installation (Developer Mode)

1. Open `chrome://extensions` in Chrome.
2. Enable **Developer mode** (top-right).
3. Click **"Load unpacked"** and select the `browser_column_select` folder.

## Supported editing areas

- `<textarea>` elements
- Elements with the `contenteditable` attribute or `document.designMode = "on"`

Confirmed working on: Gmail, Yahoo Mail, general web editors, and most other web pages.

## Known limitations

- Rectangle selection operates on visual lines (after word wrap), not logical lines.
- In rich-text (HTML) editors, cutting may occasionally alter surrounding formatting depending on DOM structure.
- Applies to all pages matching `https://*/*` and `http://*/*`.

# 矩形選択拡張 (Column Select) — 仕様書

任意のWebページの `<textarea>` および `contenteditable` 編集エリアで、サクラエディタのような矩形(カラム)選択 → コピー/カットができるようにするChrome拡張機能。

## 操作方法

1. 編集エリア内で **Alt キーを押しながらドラッグ** すると矩形範囲が青枠+黄色ハイライトで表示される。
2. **Ctrl+C** → 矩形範囲のテキストをクリップボードへコピー（各行を `\n` で連結）。
3. **Ctrl+X** → コピーと同時に削除（カット）。
4. **Delete / Backspace** → コピーせずに削除。
5. **Esc** または別の場所でクリック → 矩形選択を解除。

カット/削除は `document.execCommand` を使うため、**Ctrl+Z** で元に戻せる。

### キーボード矩形選択

- **Alt+Shift+矢印** または **Ctrl+Alt+矢印** でカーソル位置を起点に矩形を拡張できる。
- WindowsでAlt+Shiftがそのまま使えない場合はCtrl+Alt+矢印を使う。

### カラム挿入

矩形選択中に通常文字キーを押すと、選択した各行の左端に文字が挿入される。IME（全角）入力にも対応。

### 置換ダイアログ（Ctrl+H）

- 矩形選択中 → 選択した各行の範囲内のみ置換
- テキスト選択中 → 選択範囲内のみ置換
- 何も選択していない → 本文全体を置換

## インストール方法（開発者モード）

1. Chromeで `chrome://extensions` を開く。
2. 右上の「デベロッパーモード」をONにする。
3. 「パッケージ化されていない拡張機能を読み込む」をクリックし、`browser_column_select` フォルダを選択する。

## 対応する編集領域

- `<textarea>` 要素
- `contenteditable` 属性または `document.designMode="on"` の編集領域

動作確認済み: Gmail、Yahoo Mail、その他一般的なWebエディタ。

## 制限事項

- 矩形選択は「見た目の行（折り返し後）」単位で動作する。
- リッチテキストエディタでカットすると、DOM構造によっては周囲の書式に影響が出る場合がある。
- 動作対象は `https://*/*` および `http://*/*`（すべてのWebページ）。
