Column Select — Rectangular / Block Selection

## Short description (132 chars max)
Column mode / block selection for Gmail, Yahoo Mail & any webpage. Alt+Drag to select a vertical box, then copy, cut, paste or replace.

## Detailed description (English)

**Column Select** brings **column mode editing** (also known as **block selection**, **rectangular selection**, **box selection**, or **vertical selection**) to your browser — the same feature you love in VS Code, Notepad++, Sublime Text, or Vim visual block mode, now available on any web page.

Perfect for cleaning up email quotes (`>` marks), editing columns of data, bulk-inserting text at the start of every line, and find & replace within a selected column only.

### Works everywhere
- Gmail compose window
- Yahoo Mail
- Any `<textarea>` (comment boxes, code editors, forms)
- Any `contenteditable` area (rich text editors, web apps)

### Features

**Mouse selection**
Hold Alt and drag to draw a rectangle. A blue border and yellow highlight show the selected region.

**Keyboard selection**
Alt+Shift+Arrow (or Ctrl+Alt+Arrow) extends the rectangle from your cursor — no mouse needed.

**Copy / Cut / Paste / Delete**
- Ctrl+C — copy each selected row, joined by newline
- Ctrl+X — cut
- Ctrl+V — paste clipboard text into every selected row at once (multi-line paste)
- Delete / Backspace — delete only
- Ctrl+Z — full Undo support

**Column insert (multi-cursor typing)**
While a rectangle is selected, type any character and it is inserted into every selected row simultaneously. Works with IME (Japanese, Chinese, Korean) input too.

**Find & Replace (Ctrl+H)**
Replace text within the rectangle only, within a text selection, or across the entire field — the scope is highlighted so you always know what will change.

### Keyboard shortcuts

| Shortcut | Action |
|---|---|
| Alt + Drag | Rectangle selection |
| Alt+Shift+↑↓←→ | Extend selection |
| Ctrl+Alt+↑↓←→ | Extend selection (alt) |
| Ctrl+C / X / V | Copy / Cut / Multi-paste |
| Delete / Backspace | Delete |
| Ctrl+H | Find & Replace in selection |
| Ctrl+Z | Undo |
| Esc | Clear selection |

### Privacy
No data is collected or transmitted. Everything runs locally in your browser. See the Privacy Policy for details.

## 詳細説明（日本語）

**Column Select** は、ブラウザ上で**矩形選択**（**列選択・ボックス選択・垂直選択**）ができるようになる拡張機能です。VS Code・サクラエディタ・秀丸・Notepad++ でおなじみのあの機能が、GmailやWebフォームでそのまま使えます。

メール返信時の引用符「>」の一括削除・一括挿入、表形式データの列編集、全行の先頭に同じ文字を挿入、選択した列の中だけ置換——そんな作業が一瞬で終わります。

### 対応範囲
- Gmail 作成画面
- Yahoo!メール
- あらゆる `<textarea>`（コメント欄・フォームなど）
- あらゆる `contenteditable` 領域（リッチテキストエディタ・Webアプリ）

### 機能

**マウスで選択** — Alt を押しながらドラッグで矩形選択。青枠と黄色ハイライトで範囲を表示。

**キーボードで選択** — Alt+Shift+矢印（または Ctrl+Alt+矢印）でカーソルから矩形を拡張。

**コピー / カット / ペースト / 削除**
- Ctrl+C — 選択した各行をコピー（改行区切り）
- Ctrl+X — カット
- Ctrl+V — クリップボードの内容を全行に同時ペースト
- Delete / Backspace — 削除
- Ctrl+Z — Undo 完全対応

**一括文字入力（マルチカーソル）** — 矩形選択中に文字を入力すると全行に同時挿入。日本語IME対応。

**検索・置換（Ctrl+H）** — 矩形の中だけ／テキスト選択の中だけ／本文全体を選んで置換。対象範囲は色付きハイライトで表示されるので安心。

### プライバシー
データの収集・送信は一切ありません。すべてブラウザ内で完結します。
